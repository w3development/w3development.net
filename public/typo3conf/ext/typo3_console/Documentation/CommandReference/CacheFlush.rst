===========
cache:flush
===========


**Flush all TYPO3 caches**

Flushes all TYPO3 caches. Opcode cache will not be flushed.



Options
~~~~~~~

`--files-only`
   Only file caches are flushed. Useful when TYPO3 is not set up or DB connection can not be established.

- Accept value: no
- Is value required: no
- Is multiple: no
- Default: false

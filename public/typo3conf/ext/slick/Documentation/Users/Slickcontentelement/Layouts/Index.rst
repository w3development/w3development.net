Layouts
=======


Default
-------

.. figure:: slickcontentelement-layout-0.jpg
	:width: 800px
	:alt: Default layout

	Default layout

Properties:

* Content box 
	* is placed on the bottom
	* has full width
	* has a transparent white background


Layout 1
--------

.. figure:: slickcontentelement-layout-1.jpg
	:width: 800px
	:alt: Layout 1

	Layout 1

Properties:

Same as Layout 1 but

* Content box
	* width is set to auto
	* alignment is left


Layout 2
--------

.. figure:: slickcontentelement-layout-2.jpg
	:width: 800px
	:alt: Layout 2

	Layout 2

Properties:

Same as Layout 1 but

* Content box
	* width is set to auto
	* alignment is right
	* has a transparent white background


Layout 3
--------

.. figure:: slickcontentelement-layout-3.jpg
	:width: 800px
	:alt: Layout 3

	Layout 3

Properties:

* Content box 
	* is placed on the bottom
	* header and text has a shadow


Controlling
-----------

* Height of the content box is controlled by the Flexform / Plugin > field: height
* Layout is controlled by the Flexform / Plugin > field: layout


CSS
---

If you like to adapt the CSS for a layout, please refer to the basic CSS file at

* `Resources/Public/CSS/default.css <https://gitlab.die-netzmacher.de/typo3/slick/blob/master/Resources/Public/CSS/default.css>`_
